# sina-lotto-lab

一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或具体网站，仅作为页面的结构化存储与展示空间。

## 项目简介

`sina-lotto-lab` 旨在提供一个整洁、可维护的静态页面集合。  
各个 HTML 文件相互独立，彼此之间无依赖关系，适合存放各类小型前端工具、演示页面、实验性组件或离线文档片段。

## 目录说明

```
sina-lotto-lab/
├── pages/          # 主要页面归档目录
│   ├── example1.html
│   └── example2.html
├── assets/         # 公共资源（CSS、JS、图片等）
├── index.html      # 仓库首页 / 导航页（可选）
└── README.md       # 本文件
```

- **pages/**：存放所有独立 HTML 页面。每个页面应自包含，避免跨页面依赖。  
- **assets/**：可放置页面间共享的样式、脚本或媒体文件，建议按类型或用途组织子目录。  
- **index.html**：可选，可作为仓库的入口页面或简单导航。

## 页面归档说明

- 所有 HTML 页面需遵循标准 HTML5 结构，确保在主流浏览器中可直接打开。  
- 页面文件名建议使用英文小写，以连字符 `-` 分隔单词，避免空格和特殊字符。  
- 如需引用外部资源（如 CDN 库、字体等），请在页面头部注明来源及版本。  
- 每个页面应包含明确的 `<title>` 标签，便于识别和索引。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，或对现有页面进行 bug 修复。  
- 添加新页面时，请确保其不违反任何法律法规，不包含恶意代码或商业推广内容。  
- 若页面涉及敏感数据或接口，请自行做好脱敏处理，仓库维护者不承担相关责任。  
- 建议在提交前使用 HTML 验证工具检查页面结构完整性。
